# Curated slicer plates (Bambu Studio .3mf)

Pre-arranged, support-painted Bambu Studio project files for the K1 print
kit — orientation, painted supports and profile settings already applied
per the [MANIFEST](../MANIFEST.md) and the
[printing guide](../../../../docs/k1/printing-guide.md).

**Curation rule: a plate lands here only after it has produced a clean
print on a real machine.** Scratch slicer projects stay untracked
(gitignored) until they earn it. Name plates `k1-<part-group>.3mf`,
e.g. `k1-arbors-escapement.3mf`, `k1-barrel-winding.3mf`.
